Project

Information

The Client

Clipboard is a challenge taken from Frontend Mentor. The challenge is to make the site responsive and mobile friendly. The page should look exactly like the provided design.

The Goal

The main goal was to make the whole website responsive. They provided me all the assets witch included all the images and icons. The goal was to make the correct layout. There was a style guide for me, it had all the color codes and font styles. The goal was to create the correct layout so that all the content fits in the right place.

Project

Result

This website is a responsive website. It uses animate.css to to create smooth animations. It is built with flex box and css grid. It uses css variables to add different types to color codes. It is a smooth slick and responsive website that can grab any users attention.
