Clipboard is a challenge taken form Frontend Mentor. The challenge is to make the site responsive and mobile friendly. The page should look exactly like the provided design.

Development

This website is a responsive website. It uses animate.css to to create smooth animations. It is built with flexbox and css grid. It uses css variables to reuse different styles. It is a smooth slick and responsive website that can grab any users attention.
