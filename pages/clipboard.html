Development

This website is a responsive website. It uses animate.css to to create smooth animations. It is built with flex box and css grid. It uses css variables to reuse different styles. It is a smooth slick and responsive website that can grab any users attention.

Responsive

This website is responsive & its made for desktop and mobile. You can check it in the browser. I have added media queries to make it responsive. The max size 1920x1080 and the mobile is for Iphone X.
