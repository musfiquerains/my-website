Overview

Responsive Onepage website

The Project

This is a website for a vr company. They needed something fresh and well looking website. The problem was they were a small company with no website for their business, so they needed someone who can create a one page landing page that is well looking with good ui/ux design. They wanted the website to be responsive and mobile friendly.

The Solution

This is an elegant website built with HTML CSS and JS. The website used css grid so it is easier to add multiple columns. This website works on smart-phones and tablet because it is very responsive. Some elements have beautiful transition effects.

Featured sections

Css grid

I have used css grid in this website for multiple columns and rows. Css grid gives us the ability to create different types of layouts in our websites. Css grid is great for responsive design.

Responsive design

I have made this website responsive. This website is ready for Iphone X, Pixle 2 XL and many other devices. I tested this in the browser inspector, you can also check this in the browser by shift+ctrl+i.
