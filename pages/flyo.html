Information

The Client

This project was taken form an online training platform called Frontend Mentor. The challenge was to complete an existing website ui design. They provided me the design for the desktop and mobile. The challenge was to make the website exactly like the design.

The Goal

The main goal was to make the whole website responsive. They provided me all the assets witch included all the images and icons. The goal was to make the correct layout. There was a style guide for me, it had all the color codes and font styles. The goal was to create the correct layout so that all the content fits in the right place.

Project

Result

At last the project was completed and it looked exactly like the design. It felt very crisp and responsive. I have used 1920x1080 size desktop in it so it is viewable in larger screen.
