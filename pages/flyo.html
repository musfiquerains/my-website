Role

UI Design/Coding

Date

Overview

Responsive Onepage Landing Page

Challenge

This project was taken form an online training platform called Frontend Mentor. The challenge was to complete an existing website ui design. They provided me the design for the desktop and mobile. The challenge was to make the website exactly like the design.

Goal

The main goal was to make the whole website responsive. They provided me all the assets witch included all the images and icons. The goal was to make the correct layout. There was a style guide for me, it had all the color codes and font styles. The goal was to create the correct layout so that all the content fits in the right place.

Tools

For this project I have used Photoshop, Sublime text, Illustrator, Inkscape, Emmet etc.

Development

I started this project in a blank HTML document and slowly worked my way through the finish. I have used flexbox to position all the elements. I have added a css library called hover.css for different types of hover effects. I have made all the blocks in HTML5.

The
Result

At last the project was completed and it looked exactly like the design. It felt very crisp and responsive. I have used 1920x1080 size desktop in it so it is viewable in larger screen.

Featured sections

Svg icons

The work section is created with beautiful svg icons. Svg icon can be scaled to any size you want. This section uses flexbox to position the elements.

Section with illustrations

The section are created with illustrations to describe something. Now days it has become a trend to put some illustrations in a website to gain attention.
