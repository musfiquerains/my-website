Responsive Onepage Landing Page

Project
Description

This was a coding project challenge taken from front-end mentor. The challenge was to complete a website design and turn it into a working website. The website needed to be responsive. It needed to be user friendly & work across mobile devices.

Development

This is a responsive landing page. This landing page is a dark theme landing page. It uses css grid for multiple columns. This website is beautiful smooth and responsive. It works across all devices. The users can view it on android or ios devices. It has testimonials section and newsletter section.

Featured sections

Svg icons

The work section is created with beautiful svg icons. Svg icon can be scaled to any size you want. This section uses flexbox to position the elements.

Section with illustrations

The section are created with illustrations to describe something. Now days it has become a trend to put some illustrations in a website to gain attention.
