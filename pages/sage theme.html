Overview

Responsive Multipurpose Onepage website

Project
Description

This project was taken from IdeaUi designs. This website is designed for many different uses. There was a Psd design provided to the developer to make it into a working website. The website needed to be smooth & responsive to the users.

Development

This website is fully customizable and responsive. I have used css grid in this website. Flex-box has also been used to position the elements. This website has reusable button classes so it is easier for anyone to add any buttons. It has beautiful slider functionalities added. Hover elements has beautiful transitions. Hover css has been added for many different hover effects. This website can be customized in many different ways.

Featured sections

Work Section

The services section is made for anyone who wants to add different types of services to show what they do. It has font awesome icon for different types of icon sets.

Blog section

This is a css grid blog section. This section show the featured news or blog articles.
